# EkoMobil Auction Eligibility Matrix

| Entity | Requirement | Threshold |
| --- | --- | --- |
| Private seller | Verified identity | required |
| Private seller | Trust completion | 80%+ |
| Dealer | Verified dealer profile | required |
| Listing | Trust score | 75+ |
| Listing | VIN verification | required |
| Listing | Open review cases | none critical |
| Bidder | Account age | configurable |
| Bidder | Phone verified | required |
